# GhostTag

**看不见，直到你想看见。**

一个温柔的 Obsidian 插件，轻轻地将你笔记中的情感和思绪隐藏起来。

## 🌙 理念

有些话，我们想留下，却又不想留下。
有些情绪，我们不想每天看到，却需要在准备好时去面对。
GhostTag 拥抱这份复杂与脆弱。

- 你写下的，永远不会消失
- 但它静静地隐藏着，直到你准备好
- 它不会出现在打印或复制中
- 只有当你想看到时，它才会显现

---

## ✨ 功能

### 🎭 隐身模式
用 `%%...%%` 包裹的文字会变得透明而不可见。

### 👀 悬停显示
将鼠标移到上面，文字会柔和地浮现。

### ✏️ 点击编辑
将光标放在其中，文字会完全可见以便编辑。

### 👻 Ghost化切换
选中任意文字，一键用 Ghost 定界符包裹——或取消包裹来切换。

- **快捷键**: `Ctrl+Shift+G`（Mac 上为 `Cmd+Shift+G`）
- 也可从右键菜单和命令面板使用
- 空选也可使用——插入定界符并将光标置于其间

### 🔍 Ghost Scanner
一个侧边栏，列出当前笔记中所有的 Ghost。

- **想看时才打开**: 通过功能区图标或命令面板激活
- **不想看时隐藏**: 关闭侧边栏，它就完全消失
- **点击跳转**: 从列表中选择以导航到该 Ghost

只有当你准备好看到它们时，你的 Ghost 才会显现。
否则，它们完全隐藏——没有干扰，没有打扰。

### 🛡️ 复制保护
按下 `Ctrl+C` 时，Ghost 内容会被自动去除（可在设置中切换）。

### 🌍 多语言支持
支持 English、日本語、中文、Русский、Português、Français、Deutsch 和 Español。

### 🎨 完全可定制

- 更改定界符（`%%`、`((`、`==` 等）
- 调整不透明度和动画速度

### 🖨️ 打印和 PDF 中隐藏
Ghost 在打印和 PDF 导出时完全不可见。

### 📱 移动端支持
GhostTag 在 Obsidian 桌面版和移动版上均可使用。

## 📦 安装

### 从 Obsidian 社区插件（审核通过后）

1. 打开 Obsidian 设置
2. 前往社区插件
3. 搜索 "GhostTag"
4. 安装并启用

### 手动安装

1. 从 [Releases](https://github.com/itibo/ghost-tag/releases) 下载最新版本
2. 将 `main.js`、`manifest.json` 和 `styles.css` 复制到 `.obsidian/plugins/ghost-tag/`
3. 在 Obsidian 中启用插件

## 🚀 使用方法

### 基本用法

只需用 `%%` 包裹文字：

```markdown
今天是美好的一天 %% 但我内心其实很挣扎 %%
```

包裹的部分变得透明，悬停时会浮现。

### Ghost化切换

选中文字并按 `Ctrl+Shift+G` 用 Ghost 定界符包裹。
再次按下即可取消——无缝切换操作。

### 打开 Ghost Scanner

- 点击功能区图标（🔍）
- 或通过命令面板：**"Open Ghost Scanner"**

### 右键菜单

选中文字后右键：

- **将选中内容Ghost化**: 切换 Ghost 定界符的添加与移除
- **包含标签复制**: 复制时包含 Ghost 标记
- **去除标签复制**: 复制时移除 Ghost 内容

## ⚙️ 设置

### 定界符

自定义起始和结束标记：

- `%% ... %%`（默认）
- `(( ... ))`
- `== ... ==`
- 或任何你喜欢的

### 显示

| 设置 | 说明 |
|---|---|
| 悬停不透明度 | 悬停时 Ghost 的可见程度 |
| 编辑模式不透明度 | 光标在内时的可见程度 |
| 定界符不透明度 | 编辑模式中标记的可见程度 |
| 动画速度 | 淡入淡出过渡时间（毫秒） |

### 复制行为

| 设置 | 说明 |
|---|---|
| 复制时自动去除 | 按 `Ctrl+C` 时移除 Ghost |

## 💭 使用场景

### 1. 情感日记

```markdown
今天的会议进展顺利。
%% 但老板的态度真的让我很难过 %%
继续推进下一个项目。
```

### 2. 创意写作

```markdown
她微笑着回答。
%% 内心却在策划报复 %%
"我没事，"她说。
```

### 3. 学习笔记

```markdown
理解了量子力学的基础。
%% 不确定性原理还是搞不懂 %%
接下来学习波函数。
```

### 4. 工作笔记

```markdown
项目按计划进行中。
%% 实际上已经落后两周了 %%
下周向客户演示。
```

## 🎨 设计哲学

GhostTag 不是"完全隐藏"信息，
而是温柔地将它包裹起来。

- 存在着，但不可见
- 在那里，但不打扰
- 隐藏着，但可触及

就像心中的低语，你的情感以量子态存在——
只有当你选择去观察时，它们才会显现。
