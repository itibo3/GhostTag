# GhostTag

**見たいと思うまで、見えない。**

Obsidian ノート内の感情や思考を、優しく包み隠すプラグインです。

## 🌙 コンセプト

人には、残したいけど残したくない言葉があります。
普段は見たくないけど、見たい時もある感情があります。
GhostTag は、そんな弱さや複雑さに寄り添います。

- 書いたことは消えません
- でも、見たい時まで静かに隠れています
- 印刷やコピーでは現れません
- 自分が見たいと思った時だけ、ふわりと姿を現します

---

## ✨ 特徴

### 🎭 ステルスモード
`%%...%%` で囲んだテキストは、通常時は透明で見えません。

### 👀 ホバーで可視化
マウスを乗せると、ふわりと浮かび上がります。

### ✏️ クリックで編集
カーソルを置くと、はっきり見えて編集できます。

### 👻 Ghost化トグル
テキストを選択して、一瞬で Ghost デリミタを付け外しできます。

- **ショートカット**: `Ctrl+Shift+G`（Mac では `Cmd+Shift+G`）
- 右クリックメニューやコマンドパレットからも使用可能
- 選択なしでも使える — デリミタを挿入してカーソルを間に配置

### 🔍 Ghost Scanner
ノート内の全ての Ghost を一覧表示するサイドバー。

- **見たい時だけ開く**: リボンアイコンやコマンドで起動
- **見たくない時は隠れている**: 閉じればサイドバーごと消える
- **クリックでジャンプ**: 一覧から選んでその場所へ移動

あなたが見たいと思った時だけ、全ての Ghost が姿を現します。
普段は完全に隠れているので、邪魔になりません。

### 🛡️ コピー保護
`Ctrl+C` でコピーする際、Ghost 部分を自動除去（設定で ON/OFF 可能）。

### 🌍 多言語対応
English, 日本語, 中文, Русский, Português, Français, Deutsch, Español に対応。

### 🎨 カスタマイズ可能

- デリミタ（`%%`, `((`, `==` など）を自由に変更
- 不透明度、アニメーション速度を調整

### 🖨️ 印刷・PDF で非表示
印刷や PDF 出力時には完全に非表示になります。

### 📱 モバイル対応
GhostTag は Obsidian のデスクトップ版・モバイル版の両方で動作します。

## 📦 インストール

### Obsidian コミュニティプラグインから（審査通過後）

1. Obsidian を開く
2. 設定 → コミュニティプラグイン
3. 「GhostTag」を検索
4. インストール & 有効化

### 手動インストール

1. [Releases](https://github.com/itibo/ghost-tag/releases) から最新版をダウンロード
2. `main.js`、`manifest.json`、`styles.css` を `.obsidian/plugins/ghost-tag/` にコピー
3. プラグインを有効化

## 🚀 使い方

### 基本的な使い方

テキストを `%%` で囲むだけ：

```markdown
今日はいい日だった %% でも本当は辛かった %%
```

囲んだ部分は透明になり、ホバーで浮かび上がります。

### Ghost化トグル

テキストを選択して `Ctrl+Shift+G` を押すと、Ghost デリミタで囲みます。
もう一度押すと解除 — シームレスなトグル操作です。

### Ghost Scanner を開く

- リボンアイコン（🔍）をクリック
- またはコマンドパレットから **「Open Ghost Scanner」**

### 右クリックメニュー

テキストを選択して右クリック：

- **選択範囲をGhost化**: Ghost デリミタの付け外しをトグル
- **タグ付きコピー**: Ghost も含めてコピー
- **タグなしコピー**: Ghost を除いてコピー

## ⚙️ 設定

### デリミタ

開始記号・終了記号を自由に変更できます：

- `%% ... %%`（デフォルト）
- `(( ... ))`
- `== ... ==`
- その他、お好みで

### 表示設定

| 設定項目 | 説明 |
|---|---|
| ホバー時の不透明度 | マウスを乗せた時の濃さ |
| 編集モードの不透明度 | カーソルを置いた時の濃さ |
| デリミタの不透明度 | 編集モード時の記号の濃さ |
| アニメーション速度 | フェードの速さ（ms） |

### コピー動作

| 設定項目 | 説明 |
|---|---|
| 自動除去 | `Ctrl+C` 時に Ghost を自動除去するか |

## 💭 使用例

### 1. 日記での感情記録

```markdown
今日のミーティングは順調だった。
%% でも上司の態度が本当に嫌だった %%
次のプロジェクトに向けて準備を進める。
```

### 2. 創作活動

```markdown
彼女は笑顔で答えた。
%% 内心では復讐を企んでいた %%
「大丈夫よ」と彼女は言った。
```

### 3. 学習ノート

```markdown
量子力学の基礎を理解した。
%% まだ不確定性原理のところが分からない %%
次は波動関数について学ぶ。
```

### 4. 仕事のメモ

```markdown
プロジェクトは予定通り進行中。
%% 実際は2週間遅れている %%
クライアントへの報告は来週。
```

## 🎨 デザイン哲学

GhostTag は「完全に隠す」のではなく、「ふわっと包む」ことを目指しています。

- 存在するけど、見えない
- そこにあるけど、邪魔にならない
- 隠れているけど、アクセスできる

まるで心の中のささやきのように、あなたの感情は量子状態で存在します —
あなたが見たいと思った時だけ、姿を現します。
